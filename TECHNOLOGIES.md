# QLM Mobile Suite - نظام الأمين الشرعي

## التقنيات المستخدمة

### الإطار الأساسي

| التقنية | الوصف |
|---------|-------|
| **Flutter** | إطار عمل Google لبناء تطبيقات متعددة المنصات |
| **Dart** | لغة البرمجة المستخدمة مع Flutter |

### التصميم والواجهة

| التقنية | الوصف |
|---------|-------|
| **Material Design 3** | نظام تصميم Google الحديث |
| **Material You** | ألوان ديناميكية تتكيف مع النظام |
| **RTL Support** | دعم كامل للعربية من اليمين لليسار |

### إدارة الحالة (State Management)

| التقنية | الوصف |
|---------|-------|
| **Provider** | إدارة الحالة الموصى بها من Flutter |
| **ChangeNotifier** | للتحديثات التفاعلية |

### الشبكة والـ API

| التقنية | الوصف |
|---------|-------|
| **Dio** | مكتبة HTTP متقدمة |
| **Interceptors** | للمصادقة وإدارة الأخطاء |

### التخزين المحلي

| التقنية | الوصف |
|---------|-------|
| **SQLite (sqflite)** | قاعدة بيانات محلية |
| **Hive** | تخزين سريع للبيانات المؤقتة |
| **flutter_secure_storage** | تخزين آمن للـ tokens |

## هيكل المشروع

```
lib/
├── core/                    # البنية التحتية المشتركة
│   ├── theme/              # الألوان والخطوط والتصميم
│   ├── widgets/            # المكونات المشتركة
│   ├── network/            # API Client
│   └── services/           # الخدمات (Token, Sync)
│
├── features/               # الميزات (Clean Architecture)
│   ├── auth/              # تسجيل الدخول
│   ├── admin/             # واجهة المدير
│   ├── guardian/          # واجهة الأمين الشرعي
│   └── registry/          # السجلات والقيود
│
└── app.dart               # نقطة البداية
```

## مكونات التصميم

### الألوان (Material 3)

- **Emerald Green** (#10B981) - لون الأمين الشرعي
- **Royal Blue** (#1E40AF) - لون المدير
- **Amber Gold** (#D97706) - لون الموظف

### الخطوط

- **Tajawal** - خط عربي احترافي (محلي، لا يحتاج إنترنت)

### المكونات المشتركة

- `MainAppBar` - شريط علوي مع إشعارات وقائمة مستخدم
- `RoleBottomNav` - تنقل سفلي حسب دور المستخدم
- `StatusBadge` - شارات الحالة (نشط، منتهي، معلق)
- `StatCard` - بطاقات الإحصائيات
- `LoadingShimmer` - تأثيرات التحميل
- `EmptyState` - شاشات الحالة الفارغة
